PGURE-SVT

Singular value thresholding for denoising video microscopy data

PGURE-SVT is an algorithm designed to denoise image sequences acquired in microscopy. It exploits the correlations between consecutive frames to form low-rank matrices, which are then recovered using a technique known as nuclear norm minimization. An unbiased risk estimator for mixed Poisson-Gaussian noise is used to automate the selection of the regularization parameter, while robust noise and motion estimation maintain broad applicability to many different types of microscopy.

Installation

Dependencies

  • CMake 2.8+
  • Armadillo 6.400+
  • NLopt 2.4.2+
  • LibTIFF - for the standalone executable only

Compiling from source

To build PGURE-SVT, first unpack the source and create a directory named build: 

$ tar -xzf pgure-svt.tar.gz
$ cd pgure-svt
$ mkdir build

Navigate to the newly-created build directory to compile the source. This will generate a shared library in the /usr/local/lib directory. It will also install the Python wrapper by running python setup.py install.

$ cd build
$ cmake ..
$ make
$ sudo make install

The standalone executable is not built by default, as it requires LibTIFF to be installed. To install the PGURE-SVT executable into /usr/local/bin, use:

$ cmake -DBUILD_EXECUTABLE=ON ..
$ make
$ sudo make install

Using PGURE-SVT with Python

An example file, demo.py, is provided to show users how the algorithm can be linked with the HyperSpy multi-dimensional data analysis toolbox, which provides a number of useful features including data visualization and data import from a number of microscopy file formats.

import numpy as np
from pguresvt import pguresvt

# Import HyperSpy
import hyperspy.api as hs
from hyperspy.hspy import *
import matplotlib.pyplot as plt

# Load example dataset
movie = hs.load("./test/examplesequence.tif")
X = np.transpose(movie.data)

# Initialize the algorithm
svt = pguresvt.SVT(patchsize=4,
                   length=15,
                   optimize=True,
                   tol=1e-6)

# Run the algorithm on the data X
svt.denoise(X)

# Plot the denoised data Y
im = hs.signals.Image(np.transpose(svt.Y))
im.plot(navigator='slider')

Using PGURE-SVT from the command line

The PGURE-SVT standalone executable uses a simple command-line interface along with a separate parameter file. It is convenient to use on remote computing clusters. The parameter file allows the user to customize various options of the PGURE-SVT algorithm. An example file, param.svt is provided. 

$ ./PGURE-SVT param.svt

Note that the standalone executable currently only supports TIFF sequences.
