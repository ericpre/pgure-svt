Using PGURE-SVT with Python

Home > Using PGURE-SVT with Python

PGURE-SVT includes two methods for integrating the algorithm into your Python workflow. This is often convenient for importing proprietary file formats, for any pre- or post-processing of the data, and for plotting results.

A standalone executable is available, but only supports TIFF sequences.

Integration with HyperSpy

PGURE-SVT can be integrated with the HyperSpy multi-dimensional data analysis toolbox, which provides a number of useful features including data visualization and data import from a number of microscopy file formats. An example iPython notebook is provided.


  %matplotlib qt
  import hyperspy.api as hs
  from pguresvt import hspysvt

  # Load example dataset
  movie = hs.load("./examples/examplesequence.tif")

  # Take the first 15 frames, and plot the result
  movie = movie.inav[:15]
  movie.plot(navigator'slider')

  # Denoise the HyperSpy signal directly.
  svt = hspysvt.HSPYSVT(patchsize=4,
                        length=15,
                        optimize=True,
                        tol=1e-6))
  denoised_movie = svt.denoise(movie)

  # Plot denoised data
  denoised_movie.plot(navigator'slider')

Standalone Python wrapper

PGURE-SVT comes with a simple Python wrapper that accepts any NumPy array with dimensions (nx, ny, T) for denoising. For more information on the effects of the parameters, users are referred to the publication.

import numpy as np
from pguresvt import pguresvt

# Example dataset has dimensions (nx, ny, T)
X = np.random.randn(128, 128, 25)

# Initialize the algorithm
# with some parameters
svt = pguresvt.SVT(patchsize=4,
                   length=15,
                   optimize=True,
                   tol=1e-6)

# Run the algorithm on the data X
svt.denoise(X)

# Get the denoised data Y
Y = svt.Y
