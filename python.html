Using PGURE-SVT with Python

Home > Using PGURE-SVT with Python

PGURE-SVT comes with a simple Python wrapper that accepts any numpy array with dimensions (nx, ny, T) for denoising. For more information on the effects of the parameters, users are referred to the publication.

import numpy as np
from pguresvt import pguresvt

# Example dataset has dimensions (nx, ny, T)
X = np.random.randn(128, 128, 25)

# Initialize the algorithm
# with some parameters
svt = pguresvt.SVT(patchsize=4,
                   length=15,
                   optimize=True,
                   tol=1e-6)

# Run the algorithm on the data X
svt.denoise(X)

# Get the denoised data Y
Y = svt.Y

Integration with HyperSpy

PGURE-SVT can also be integrated with the HyperSpy multi-dimensional data analysis toolbox, which provides a number of useful features including data visualization and data import from a number of microscopy file formats. Instead of accepting a numpy array, this wrapper will accept a HyperSpy signal for denoising. An example iPython notebook is provided.

# Configures the plotting backend
%matplotlib qt

# Import the hyperspy API
import hyperspy.api as hs

# Import the hyperspy wrapper for PGURE-SVT
from pguresvt import hspysvt

# Load example dataset
movie = hs.load("./test/examplesequence.tif")

# Take the first 15 frames, and plot the result
movie = movie.inav[:15]
movie.plot()

# Denoise the HyperSpy signal directly.
svt = hspysvt.HSPYSVT(patchsize=4,
                      length=15,
                      optimize=True,
                      tol=1e-6))
denoised_movie = svt.denoise(movie)

# Plot denoised data
denoised_movie.plot()
